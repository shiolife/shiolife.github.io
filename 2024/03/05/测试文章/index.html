What is this

This is a script that generate a merge mod that allows you to set outfit(s) to specific region and swap/toggle them specifically within those region.

This mod required Region.ini to work. You can download it from the post below

https://gamebanana.com/tools/15459

How this mod work

If you set it up correctly here are the effect the mod will have

  1. When you enter a specific region that you have setup outfit to be changed to, your outfit will be swapped to the one that you setup for that region.
  2. If you have setup multiple outfit for a specific region, you will be able to toggle between only those outfit within that region.
  3. You will be able to swap to and toggle all outfit outside of specified regions. (Ex: if you had 5 outfit and set 1 to inazuma, 1 to liyue, all 5 will be available to be swapped and toggle to outside of inazuma and liyue)

How to setup the mod

  1. Preparation
  • ​ Download and put the folder BufferValues, which contain Region.ini file, into your GIMI’s mods folder
  • ​ Name the merge mod sub folder to begin with number in order of 0, 1, 2… format (if your naming reach two digits, name them as 00, 01, 02…10…). This means the value of the outfit in the merge.ini file will correspond to the same number you have in the folder name (0 means folder with 0, 1 means folder with 1…)
  1. Setup outfits and their region

There are two different method to do this.

・Method 1 is recommended if you will be remaking the merge mod multiple time in the future and don’t mind ugly folder name.

・Method 2 is recommended if you want your folder name to be clear and don’t intend to remake the merge mod or don’t mind redo the process in the future

  • ​ Method 1:

Step 1: Add the below 3 letter code, which corresponding to respective region, to the folder name of the mod you want to set region swap to.

IMPORTANT: Letter code must be separated from the other text with a blankspace. For each folder, multiple letter code can be used. Every letter code can be used on multiple folder

MON = mondstadt

LIY = liyue

DRA = dragonspine

INA = inazuma

ENK = enkanomiya

CHA = chasm

SUF = sumeru forest

SUD = sumeru desert

FON = fontaine

TEA = teapot

UNI = unidentified region (domain, abyss…)

CHE = chenyu

Step 2: Run the merge mod script

Step 3: When first time loading the mod (not everytime you open GIMI, just the first time with the mod loaded), press your toggle key ONCE to set the mod in its proper setting. NOTE: after this first press, every press of the toggle key will function as a normal toggle

  • ​ Method 2:

Step 1: Run the merge mod script

Step 2: Open the newly created merged.ini file and find the key related to the region you want to have toggle swap and add the outfit you want to be toggle-able in each region. The value you put for each region will be the toggle-able outfit for that region.

Step 3: Save the merged.ini

Step 4: When first time loading the mod (not everytime you open GIMI, just the first time with the mod loaded), press your toggle key ONCE to set the mod in its proper setting. NOTE: after this first press, every press of the toggle key will function as a normal toggle

Known issue

  • ​ Cannot toggle outfit in a specific region that has no outfit setup to be swapped to

How to solve: If that region that has outfit(s) to be swaped to setup and loaded in the game before, and you decide to remove all outfit from that region, you need to completely move the character mod folder (Ex: Lumine folder) out side of GIMI’s mods folder and reload GIMI once (via F10 or restart GIMI) and then put the folder back.
